# my First login page
This project is a simple Login Page created using HTML and CSS.
It is designed for beginners to understand basic web page structure and styling.

The login page includes:

Email input field

Password input field

Login button

Clean and simple design

🔹 Technologies Used

HTML5 – for structure

CSS3 – for styling

🔹 Project Files

index.html → Contains the login page structure

style.css → Contains styling for the login page

README.md → Project information

🔹 How to Use

Download or clone the project

Open index.html in any web browser

The login page will appear on the screen

🔹 Purpose of This Project

Practice HTML & CSS

Learn how to create a basic login form

Upload and manage files on GitHub

🔹 Future Improvements

Add JavaScript validation

Connect with backend

Improve UI design
